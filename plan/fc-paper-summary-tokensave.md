# Functional Consciousness (FC): Context Summary

## Core Idea

Functional Consciousness (FC) is a measurable architectural property of an agent:

> The observable capacity of a system to access and reason about internal representations of its own states.

FC is explicitly concerned with **access consciousness** and functional self-awareness, not phenomenal consciousness ("what it feels like").

The paper proposes a quantitative metric called the **Functional Consciousness Score (FCS)**.

---

# Main Definitions

## Functional Consciousness (FC)

The ability of an agent to:
1. Represent aspects of its own internal state.
2. Make these representations available to reasoning.
3. Use them to guide behavior.

## Self-Model

A self-model is any internal representation that functionally models:
- The system's own states
- Processes
- Capabilities
- Limitations

Examples:
- Body position
- Memory confidence
- Goal hierarchy
- Emotional state
- Attention focus
- Reasoning accuracy

---

# Functional Consciousness Score (FCS)

For a single self-model:

FCS = R × P

Where:
- **R** = Representational Capacity
- **P** = Reasoning Power

---

# Representational Capacity (R)

R measures how rich and accurate a self-model is.

R = B × D̄

Where:
- **B (Breadth)** = Number of independent variables tracked
- **D̄ (Depth)** = Average mutual information (bits per variable)

Interpretation:
- Breadth = how many things are modeled
- Depth = how precisely they are represented

Examples:
- Position, velocity, battery level
- Confidence, stress, memory availability

---

# Reasoning Power (P)

P measures the expansion of distinguishable future states generated by inference.

Interpretation:
- How many useful conclusions can be derived per reasoning cycle.
- Approximation of state-space expansion under reasoning.

Examples:
- Simulation
- Prediction
- Counterfactual reasoning
- Planning
- Reflection

---

# Why Multiplication?

The multiplicative form enforces two conditions:

1. Rich data without reasoning → zero consciousness.
2. Strong reasoning without persistent self-models → zero consciousness.

Examples:
- Static map: high R, P = 0 → FCS = 0
- Stateless LLM: high P, R = 0 → FCS = 0

Functional consciousness requires both representation and reasoning.

---

# Multiple Self-Models

An agent contains many self-models.

Without cross-reasoning:
FCS(agent) = Σ FCS(model_i)

With strong cross-reasoning:
- Representational capacity adds.
- Reasoning can combine across domains.
- Overall functional consciousness increases dramatically.

This explains why integrated cognition is much more powerful than isolated modules.

---

# Functional Self-Model Analysis (FSMA)

FSMA is a black-box methodology to infer self-models from behavior.

Principle:
> If an agent's output systematically depends on an internal state, the agent must possess a functional model of that state.

Examples:
- "I am tired" implies an energy/fatigue self-model.
- "I don't trust this memory" implies confidence and episodic self-models.
- "I am watching myself think" implies meta-self-awareness.

FSMA is abductive reasoning: infer the minimal internal structure required to explain observable behavior.

# Important Claim About LLMs

A stateless LLM scores zero because:
- It has substantial reasoning power.
- It lacks persistent self-models between forward passes.

An LLM embedded in an agent architecture with:
- Memory
- Reflection
- Self-monitoring
- Planning

can achieve substantial FC.

The agent scaffold, not the base model alone, is the locus of functional consciousness.

---

# Relation to Major Theories of Consciousness

FC does not attempt to solve the hard problem.

Instead, it operationalizes functional aspects shared by several theories.

## Higher-Order Theories (HOT)
Mental states become conscious when represented by higher-order states.
FC models this through self-models and meta-self-awareness.

## Global Workspace Theory (GWT)
Conscious contents are globally available.
FC makes selected internal states available to a global reasoning process.

## Attention Schema Theory (AST)
Awareness is a model of attention.
FC includes attention as a self-model target.

## Predictive Processing (PP)
Cognition minimizes prediction error.
FC quantifies predictive richness and state-space expansion.

## Integrated Information Theory (IIT)
Consciousness depends on integrated information.
FC approximates functional integration through cross-reasoning over shared self-models.

---

# Functional vs Phenomenal Consciousness

FC measures:
- Self-awareness
- Metacognition
- Introspection
- Planning over internal states

FC does NOT claim to measure:
- Subjective experience
- Qualia
- Phenomenal consciousness

The relationship between FC and phenomenal consciousness remains open.

---

# Key Insight

Consciousness-related functionality can be decomposed into:

1. Self-models
2. Representational richness (R)
3. Reasoning power (P)
4. Cross-domain integration

This turns consciousness from a binary philosophical label into a graded engineering metric.

---

# Engineering Implications

To increase FC in AI systems:
- Add explicit self-models to each subsystem.
- Preserve persistent memory.
- Expose internal states to global reasoning.
- Enable reflection and self-correction.
- Track uncertainty and confidence.
- Model goals, limitations, and ethics.

---

# AI Safety Relevance

A system that cannot model:
- Its own goals
- Capabilities
- Limitations
- Uncertainty

is inherently less reliable.

FC can serve as:
- Benchmarking metric
- Safety indicator
- Design objective

---

# Conceptual Formula Summary

Single self-model:
FCS = R × P

Representational capacity:
R = B × D̄

Agent score:
FCS(agent) = aggregation over all self-models and their cross-reasoning

---

# Key Takeaways

- Self-models are the core units of analysis.
- Functional consciousness is graded, not binary.
- Persistent self-models are necessary.
- Reasoning alone is insufficient.
- LLMs require agent scaffolding for non-zero FC.
- Humans combine many rich self-models with powerful reasoning.
- FC provides a common functional language linking major theories of consciousness.

---

# One-Sentence Summary

Functional Consciousness is a quantitative framework that measures how richly an agent models itself and how effectively it reasons over those self-models to guide behavior.
